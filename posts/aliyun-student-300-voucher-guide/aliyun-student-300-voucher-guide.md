> [!NOTE]
> 这篇整理的是 2026 年 6 月实测有效的方法。阿里云券的可用范围会变，而且是结算页说了算，所以不要把这篇当永久攻略。往后看的话，先去结算页确认能抵扣再下单。

众所周知，阿里云每年都会给大学生发 300 元代金券

听起来很大方。实际点进控制台就会发现，这张券不是 300 元现金，也不是全站随便买。官方页面写得很清楚：通过学生认证的高校学生可以领 300 元无门槛优惠券，有效期 1 年；购买范围是阿里云部分公共云产品，具体以下单购买时抵扣情况为准

而且这个券的可用范围还在不断收缩。去年能用的东西今年可能就不行了。所以我会建议你领完以后不要放着不管，尽早想好怎么花

入口在这里[学生权益领取链接](https://university.aliyun.com/course/promotion25-activity?clubTaskBiz=subTask..12655012..10273..&userCode=gv5jbukv)

## 怎么领

流程很短：

1. 有阿里云账号（支付宝注册或手机注册都行）
2. 打开学生优惠页面
3. 完成学生认证（一般是学信网验证或高校邮箱）
4. 找到「我是学生」入口，领取 300 元抵扣券

认证通过后，券会出现在你的优惠券列表里。有效期一年，从领取那天开始算

如果你还没有阿里云账号，就顺手注册一个，以后不管用不用服务器，阿里云百炼的大模型 API 也可以拿券抵扣


# 能用来干啥

## 第一种：包年大陆地区 ECS

论坛里最常见的用法是开一台中国大陆地区的 2C2G ECS，能开一年，加上这个券每年能领一个，你读多久大学基本上就能用多久

参考贴：

[https://linux.do/t/topic/2258128](https://linux.do/t/topic/2258128?u=amiya_desi)

大致思路是：

1. 进入 ECS 自定义购买页
2. 选 2C2G 左右的 e 实例
3. 地域选中国大陆（避开那些标注了「优惠」的地区，反而可能用不了券）
4. 流量计费方式选 CDT 按量
5. 结算页确认券能抵扣

ECS 自定义购买页：

[https://ecs-buy.aliyun.com/ecs/#/custom/](https://ecs-buy.aliyun.com/ecs/#/custom/)

论坛里验证过能用的关键配置：

```text
实例规格：ecs.e-c1m1.large（2C2G）
付费模式：包年包月，一年
流量计费方式：CDT 按量计费
```

选对的话，结算页应该显示 300 元以下，券直接抵扣，不用额外掏钱

如果你那边显示用不了券，换个地域试试。论坛里有人青岛能用，有人华南能用，不同时期规则不一样。重点是：看支付页，不要猜

开通 CDT 后，每月有 20GB 左右的免费流量。挂博客、机器人、小 API 完全够用，但别拿它做大流量下载站、图床外链、视频分发

买完第一件事不是装面板，是去设置费用预警。CDT 按量的意思是超出免费额度后会产生费用。设一个阈值报警，哪怕 1 块钱也报，这样被攻击或者配置错误时能及时发现

## 第二种：轻量应用服务器

如果你不想研究 ECS、安全组、弹性公网 IP、CDT 这些东西，轻量应用服务器更像「已经打包好的小 VPS」

学生优惠页面往下翻，或者直接在页面里搜「轻量应用服务器」：

[https://university.aliyun.com/buycenter/](https://university.aliyun.com/buycenter/)

目前看到的几条路线：

**A. 直接 0 元开半年 2C2G 轻量**

论坛里有人确认可以 0 元购轻量 2C2G 200M 带宽半年。优点是不限流量（带宽固定），不用担心 CDT 超额。缺点是只有半年

**B. 云小站 38 元抢购 + 券续费**

先在云小站定时抢购 38 元一年的轻量应用服务器：

[https://www.aliyun.com/minisite/goods](https://www.aliyun.com/minisite/goods)

据说比较好抢。抢到后用 300 元券续费半年。这样加起来一年半，总花费 38 元。

这个方案要小心两点：

- 活动价和学生券不一定能叠加。以你付款时的实际显示为准
- 续费时如果券不能抵扣，那 38 元就只买了一年，不是一年半

**C. 试用 ECS 转正**

这是我自己走的路线。阿里云有一个免费试用入口：

[https://free.aliyun.com](https://free.aliyun.com)

可以试用 2C2G 的 ECS 三个月。试用到期前或到期后，用 300 元券转包年，这样算下来大概能用一年三个月到更久

我个人更喜欢轻量给新手练手。不是因为它更高级，而是因为它少一点选项，少一点把自己绕晕的机会

## 第三种：非大陆轻量

非中国大陆地区就更玄学一点

论坛里的反馈是，非大陆 ECS 基本不好用券，但部分轻量应用服务器、部分地域、部分配置可能可以。比如有人提过 2C2G 通用型买 6 个月，或者 2C0.5G 买一年，总价 336 元，用券后还要补 36 元；也有人选择按月续

这里我不建议把具体地域写死

香港、新加坡、日本、美国、德国、英国、东南亚，这些地区的券规则可能今天能用，明天就不能。你真要海外机器，就把配置放进购物车，看支付页

如果你只是为了搭博客，Cloudflare Pages 已经够用了

## 第四种：无影云电脑和灵豆

阿里云学生优惠页里现在也能看到无影相关入口。

这个方向适合两类人：

- 想体验云电脑，偶尔跑跑软件或者游戏
- 想临时租 GPU 算力，不想自己买显卡

论坛里有人整理过无影灵豆和无影灵构平台的玩法，比如用灵豆开 RTX5880 48GB 显存的 GPU 环境：

[https://linux.do/t/topic/2219866/60](https://linux.do/t/topic/2219866/60)

这类帖子更适合当线索看。真正付款前，还是要回到阿里云结算页确认灵豆、地域和时长

这类玩法的好处是爽，坏处是它更像按量烧点数。你如果只是想长期挂一个服务，它不如服务器。你如果是科研党、模型党、临时要显存，那就值得看看

## 第五种：百炼和大模型 API

如果你已经有服务器，或者对云服务器没兴趣，300 元拿去试大模型 API 也可以

学生优惠页里有大模型服务、AI 应用开发、阿里云百炼的入口。云小站也能看到百炼、Token Plan、HappyHorse 等模型相关活动：

[https://www.aliyun.com/minisite/goods](https://www.aliyun.com/minisite/goods)

如果你想让代金券抵扣，优先选阿里云官方直供、官方控制台里按量计费的模型。第三方中转、市场商品、特殊活动包，可能就不走这张券

可以拿来试：

- 通义千问系列（Qwen）
- DeepSeek 相关 API
- GLM / Kimi 等平台里能选到的官方模型
- HappyHorse 这类视频模型，前提是确认结算页能抵扣

关键是调用时选「官方直供模型」。百炼控制台里有些模型标注了「三方」或「市场」，那些不一定能用这张券抵扣

300 元能调多少次？这取决于你选的模型和输入长度。通义千问 Turbo 这类便宜的，300 元可以调非常多次。如果是高端大模型，就会贵很多。先从便宜模型试起，把流程跑通再说

这比买一台闲置服务器更适合「我就是想拿来试 API」的人

## 第六种：白嫖 ACA 大模型工程师认证

阿里云还有一个骚操作：用 300 元券直接抵扣 ACA 大模型工程师认证费用。

原价 600 元的认证，学生优惠价刚好 300 元，用券后 0 元

参考帖：

[https://linux.do/t/topic/2456810](https://linux.do/t/topic/2456810)

大致流程：

1. 完成学生认证，领取 300 元券
2. 在学生优惠页往下滑，找到「大模型认证」栏目
3. 点击「立即购买」，用券抵扣

这个认证是阿里云官方的职业岗位认证，面向初学者。虽然含金量不高，但对学生来说，0 元拿一张证书、顺便学一遍大模型基础知识，不亏

**适合谁：**

- 简历需要补充认证栏
- 想系统学习大模型基础概念
- 已经有服务器或者对服务器没兴趣

**不适合谁：**

- 已经在做大模型开发的人（太基础）
- 更想拿券换服务器的人（机会成本）

如果你的券快过期了，又暂时用不到服务器和 API，拿来换这个认证也不错

## 我会怎么选

如果是第一次玩云，我会这样排：

1. 先领券
2. 看 ECS 或轻量有没有 2C2G 能用券
3. 能 0 元或少量补差价拿服务器，就先拿服务器
4. 服务器到手后立刻做 SSH、安全组、费用预警
5. 再跑 Vaultwarden、ntfy、Gatus、机器人、笔记同步之类的小服务

如果你已经有 VPS，那就别为了花券再买一台吃灰机器。拿去试百炼 API 或无影灵豆，反而更像在用钱

## 下单前检查

别急着付款，先看这几个：

- 支付页有没有出现 300 元券抵扣
- 订单是不是包年包月，还是按量付费
- 到期后会不会自动续费
- 流量、带宽、公网 IP 怎么算钱
- 是否有安全组默认放行风险
- 是否能设置费用预警和额度报警
- 退款后券还能不能重新用于同品类

尤其是最后一条。官方规则里提到，使用代金券的订单退款后，退回的代金券对应品类不可二次使用。不要拿它乱试单
