# 网易 UU 远程
官网 [https://uuyc.163.com/](https://uuyc.163.com/)

如果你只是想远程连一下自己电脑，这个确实挺省事，装完登录绑定设备就能用

但不要随便给别人开无人值守

# File Converter
网页 [https://file-converter.io/](https://file-converter.io/)

一个在线文件转换站，图片、文档、音频、视频之类的格式转换都能拿它顶一下

临时救急很好用，但敏感文件别往这种网站传

# LocalSend
官网 [https://localsend.org/zh-CN](https://localsend.org/zh-CN)

这个拿来跨端传文件很舒服，Windows、安卓、Linux 这些都能互传，局域网内直接发

如果你经常微信传文件给自己，建议装一个试试

# Obsidian
官网 [https://obsidian.md/](https://obsidian.md/)

本地笔记软件，笔记就是普通 Markdown 文件，这点我一直挺喜欢

我现在很多文章也是先在 Obsidian 里写，再慢慢整理到博客

# OBS
官网 [https://obsproject.com/](https://obsproject.com/)

录视频和开直播基本绕不过它，免费，而且足够能打

第一次装好后记得先录十秒看看画面和麦克风，不然很容易白录

# Aseprite
官网 [https://www.aseprite.org/](https://www.aseprite.org/)

画像素画和做像素动画很好用，也是很多人默认会提的一款

官方安装包是付费的，介意的话可以自己编译，比如用这个仓库[a1393323447/aseprite-builder](https://github.com/a1393323447/aseprite-builder)
