# 开头
既然要继续游戏开发了，在开发前先整理一下手中的战备吧！把我项目中常用的插件整理出来，放在这里，[Godot Asset Store](https://store.godotengine.org/)，目前官方最新的插件商店，也可以在这里找到很多有用的插件，然后再补充一个国内的Godot社区，[开源项目 - GodotHub](https://godothub.com/oss)，这里面有支持taptap和微信抖音小游戏的插件

# DialogueManager
[nathanhoad/godot_dialogue_manager: A powerful nonlinear dialogue system for Godot](https://github.com/nathanhoad/godot_dialogue_manager)，我最喜欢用的对话插件，虽然也有其他很优秀的比如[dialogic-godot/dialogic: 💬 Create Dialogs, Visual Novels, RPGs, and manage Characters with Godot to create your Game!](https://github.com/dialogic-godot/dialogic)和国内GodotHub的[Konado](https://godothub.com/oss/konado/zh/2.4/)，不过站长最习惯用的就是这个了，这个能够更简单的实现高度自定义化

# SceneManager
[m-canton/godot-scene-manager: Scene Manager Plugin for Godot.](https://github.com/m-canton/godot-scene-manager)，站长用的是这个版本，个人感觉还是挺好用的，我记得github上面还有两个百star的scene-manager插件，下次看看

# RichText2
[RichTextLabel2 - Godot Asset Library](https://godotengine.org/asset-library/asset/3254)
这个我习惯结合上面的DialogueManager里的对话使用，能够做出非常酷炫的各种文字效果，下次补一张gif图在这里

# PhantomCamera
幽灵相机除了臃肿（当然可以自己剪切掉一些），剩下的就只有方便好用了，[ramokz/phantom-camera: A Camera addon for Godot 4. Inspired by Cinemachine.](https://github.com/ramokz/phantom-camera)，地址在这里

# SoundManager
如果你的游戏没有到要上FMod这种程度但是又想更好的控制音频，那么[nathanhoad/godot_sound_manager: A simple music and sound effect player for the Godot Engine](https://github.com/nathanhoad/godot_sound_manager)会是你很好的选择

# EnhanceSaveSystem
王婆卖瓜，自卖自夸（雾），这个是站长自己氛围结合人工编程出来的存档插件，广泛运用于自己的项目中，个人感觉还是比较方便的模块式存档系统，并且比较方便的插入hook之类的，也支持各种加密存档[Amiyadesi/enhance_save_system](https://github.com/Amiyadesi/enhance_save_system)

# GodotSteam
[Welcome To GodotSteam - GodotSteam](https://godotsteam.com/)，大名鼎鼎的GodotSteam插件，可以说想上steam的话这个肯定是要研究研究了

# 最后
随着unity的不断神秘操作，导致Godot在近几年变得愈发大众，我觉得也是值得的；这个开源引擎再当今的AI时代也是更加出彩，因为它和AI的相性非常的高；总之如果你喜欢开源自由，或者就是想要尝试一个新的引擎，Godot肯定是一个很好的选择；而如果你想要用来做一些作品的话，那么合理的借助插件的力量也是理所应当的！
