# 插件是什么

![[Pasted image 20260703114150.png|width=560|align=center|caption=浏览器插件入口]]
点击右边的那个碎片就可以去插件商店下载了，浏览器插件简单来说就是一段额外的插入页面的脚本，所以能做到很多事情，所以来路不明的插件最好也不要下载，尽量下载一些比较正规的，商店里的也够用了

# AdGuard
![[Pasted image 20260703123311.png|width=520|align=center|caption=AdGuard 插件面板]]
在官方插件商店既可下载，超级好用的去广告插件，浏览器必备堪称，下载了后就可以一直开着不动了基本上

# Tampermonkey
大名鼎鼎的油猴脚本想必不用多说，这相当于插件中的自定义插件了，[https://www.youxiaohou.com/](https://www.youxiaohou.com/)，推荐可以通过这个网站里面的资源下载各种脚本和其他，然后这里给一个我用的比较多的[hmjz100/LinkSwift: 一个基于 JavaScript 的网盘文件下载地址获取工具。基于【网盘直链下载助手】修改 ，支持 百度网盘 / 阿里云盘 / 中国移动云盘 / 天翼云盘 / 迅雷云盘 / 夸克网盘 / UC网盘 / 123云盘 八大网盘](https://github.com/hmjz100/LinkSwift)，还有其他各种方便的脚本之类的，以及作为学生党，用这个能做的事也很多，具体就不说了嘻嘻

# 沉浸式翻译
![[Pasted image 20260703121638.png|width=560|align=center|caption=沉浸式翻译插件]]
可以在插件商店找到，用于做双语翻译之类的很舒服，自带的免费也不错，如果要求比较高的话可以充值一些deepseek用deepseek v4 flash来翻译也能支持很久了，或者就去找一些有deepseek的公益站之类的，比如我的这篇文章里就有[[freeapi-glm-kimi-cc-switch|一些目前的免费API]]

# Bitwarden
![[Pasted image 20260703121438.png|width=560|align=center|caption=Bitwarden 插件]]
也可在插件商店找到，自带的功能就可以实现保存身份和银行卡和密码账户还有通行密钥，并且可以自动填写，可以说是互联网混迹必备了，越早接触越好，接触了就离不开了，然后进阶的2FA保存推荐可以自己搭建一个vaultwarden[[selfhost-vaultwarden-on-2c2g|自己搭建一个vaultwarden]]，或者找一个，比如[[disroot-open-services-note|Disroot介绍]]这这个组织就有免费的vaultwarden

# GlobalSpeed
![[Pasted image 20260703121741.png|width=560|align=center|caption=GlobalSpeed 插件]]

![[Pasted image 20260703122818.png|width=560|align=center|caption=SimilarSites 插件]]
