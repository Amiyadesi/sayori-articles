# 互联网社区记录 1：Linux.do NodeLoc和NodeSeek
我一开始接触社区，还是4月份的时候因为这种事件[[2026-04-06]]，然后接触了[Doki Chinese Club](https://forum.monika.love/)，不过这一篇，我也暂时不打算讲这个，讲讲我目前有账号和收获资源比较多的两个社区吧
Linux do和Nodeloc
## Linux.do

入口：[https://linux.do/](https://linux.do/)（需要魔法。但是可以通过修改DNS直连）

佬友说得好，学AI，上L站，Linux do这个论坛，外界对其众说纷坛，神秘莫测，而小编我实地进去考察了一个多月后，打算在这里给出我的调查报告吧

L站的内部规则相对来说比较严格，具体可以参照这个[https://linux.do/guidelines](https://linux.do/guidelines)，如果新人进去了，记得先完成bot的新手任务，确定升级到了一级再去发言哦，而且要熟读这里的规则，尤其是`非必要不抽奖`，然后基本上就可以开始愉快的刷帖发帖升级之旅了

Linux do的一大特点就是有很多大佬开AI公益站，还有一些公益站接入了linxu do登录，比如any就可以用L账号注册一个新的，如果你还有教育邮箱的话基本就可以双持了，然后还有很多站内的大佬开的各种公益站，关于这些，如果你有一个账号，可以在[公益站列表贴](https://linux.do/t/topic/2140307)里面看到

然后福利羊毛里面，也是福利很多了，虽然大部分都是临时的，有时候一些最新的羊毛信息也会在这里出现，还是很有收获的

开发调优就比较技术向了，里面一般有很多AI和编程相关的问题帖和技术帖，而且一般都比较新，
[手把手教你搭建中转站 VPS+CLI Proxy API（CPA）+NEW API+Docker+Nginx+Cloudflare+SSL+域名配置 保姆级教程]([https://linux.do/t/topic/2140889](https://linux.do/t/topic/2140889?u=amiya_desi))，这个就是一个比较好的技术贴了

文档共建里面有很多长篇的有意思的教程，就比如这个[新人教程汇总](https://linux.do/t/topic/2434230?u=amiya_desi)，比我讲的全面多了（笑），那个修改DNS直连的里面也有，然后还有这个[[超详细教学] 教你从零开始，入坑域名、云服务器并部署 New API + Open WebUI！](https://linux.do/t/topic/188813?u=Amiya_desi),也是一个比较全面的知识，我也打算做一个类似的（笑）

搞七捻三里面就有很多日常和各种好玩的帖子了，感觉就有点像早期的互联网(虽然从未经历过，但是别人都是这么说的)的感觉呢！

新人注册后，推荐登录一下[https://credit.linux.do/](https://credit.linux.do/)，这个ldc就是一种社区内的积分，可以用来换取各种社区内的服务啥的

如果你不想看到广告，可以自主选择加入[https://linux.do/g/g-adsfree](https://linux.do/g/g-adsfree)，以及如果要发布金钱相关的，也需要加入这里[https://linux.do/g/g-trade](https://linux.do/g/g-trade)

总结，L站以比较严格的方式，提高了回复和发帖的质量，当然有时候也会比较难受，如果你能够好好遵守[https://linux.do/guidelines](https://linux.do/guidelines)，并且到达二级（一级过久不活跃会被删号，二级就不会），这样的话在L站基本也能混的挺好的，能够时不时的获得各种信息和知识和公益，也有了一个可以发帖分享和宣传自己的新地方，一个AI时代下的高质量的人类信息渠道

## NodeLoc

入口：[https://www.nodeloc.com/](https://www.nodeloc.com/)

NodeLoc相对来说，规则没有这么多，~~听说有些像早期的L站~~，里面的福利和其他也有很多呢，具体可以看看这个[整理了NL所有相关站点](https://www.nodeloc.com/t/topic/94649)

Nodeloc的规则在这里[https://www.nodeloc.com/about](https://www.nodeloc.com/about)，其实也可以看出除了那几个绝对违禁的点外，其他基本没啥规则

Nodeloc没有回帖字数限制，比较自由，同时每个人也可以创造和加入各种子论坛，什么域名日常运动主机sim卡AI之类的，同时这里也不禁止公开的AFF，这是相对于Linux do来说有所不同的地方

并且这里的站长比较活跃在评论区，感觉还挺友善的

Nodeloc也有那种类似社区积分的东西，叫做电池，每天右上角签到就可以获取，也可以通过水帖子获取之类的，也能用于在社区内买东西之类的

总结，Nodeloc作为一个比较新的小站点，相对来说也更加自由，当然也导致了回复的质量一般来说都比较普通，不过管理也有在调整，也是一个正在发展的社区，并且知识和福利啥的也有很多，也是一个高质量的信息渠道呢

## NodeSeek

入口：[https://www.nodeseek.com/](https://www.nodeseek.com/)

[我们论坛有什么必看的福利和服务啊](https://www.nodeseek.com/post-822458-1)，站长进去就发过一个帖子询问了一下啊，这里基本就是纯正的mjj聚集地了，大家交流的都是各种服务器相关的知识和其他，成员们基本也都有tg号，帖子里面也有佬说可以当成暗黑版百度来搜索hh

如果有想要讨论服务器相关，以及想要获取一些情报信息的话可以尝试注册看看
