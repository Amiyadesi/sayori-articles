# About

这里是 Amiya_desi 的个人博客。

关于我: 目前19岁大学生，梦想是成为独立游戏开发Master，最喜欢的游戏类别是 meta 独立游戏，也想亲手做出一个有“屏幕外一层意义”的 meta 独立游戏。平时还喜欢折腾东西，比如说这个博客，今后会在这里分享一些游戏开发、学习、工具和日常思考等内容，我期待数据能拟合出我，帮我解决那些我一个人想不明白的东西，所以我要把这些都记录下来

如果你在网上看到我到处乱冲，或者又在摸鱼，可以顺手提醒我一句：该学习了

目前能公开放出来的作品和项目主要在 GitHub：

- [WakeUpAtTheBorder](https://github.com/Amiyadesi/Wake-up-at-the-border)：Godot 2D 横版叙事游戏，GodotHub 第三届开发大赛二等奖
- [TimeRuner](https://github.com/Amiyadesi/time-runner)：C# 动作平台跳跃，5 秒时间回溯，034GameJam 最佳时间题材游戏奖
- [EnhanceSaveSystem](https://github.com/Amiyadesi/enahnce_save_system)：Godot 存档插件，支持 AES-GCM、gzip、原子写入和版本迁移
- [Mio's Windowanderer](https://github.com/Amiyadesi/ThatGameJam) / [少女们的假面舞会](https://github.com/Amiyadesi/ggj)：GameJam 里做过的多窗口解谜和文字解谜尝试
