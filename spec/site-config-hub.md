# 博客站点配置入口

以后改博客站点信息，先从这里跳：

- [spec/about.md](about.md)：关于页正文
- [spec/friends.md](friends.md)：友链页正文和申请说明
- [site/profile.json](../site/profile.json)：头像、昵称、简介、社交链接
- [site/navigation.json](../site/navigation.json)：导航栏
- [site/banner.json](../site/banner.json)：首页 Banner 和文案
- [site/announcement.json](../site/announcement.json)：公告栏
- [site/music.json](../site/music.json)：音乐播放器配置
- [friends/](../friends/)：友链卡片
- [assets/](../assets/)：头像、Banner、音乐、友链头像等素材

写文章还是去：

- [posts/](../posts/)

预览博客就点左侧插件按钮 `启动博客预览`。再点一次就是 `停止博客预览`。
